# Screenshots

Drop real UI captures here for the root README:

- `overview.png` — Dashboard / Overview
- `project.png` — a project detail or run log view

Capture from a running local UI (`http://localhost:5173` in dev, or
`http://127.0.0.1:4400` in production mode). Redact anything sensitive in
logs before committing.

Do not commit fabricated mockups. Prefer PNG or WebP under ~500KB each.
